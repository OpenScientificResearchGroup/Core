## 1.2 初始化和销毁

初始化和销毁内核的方法直接抄 [README.md](../../README_CN.md) 里的使用示例就可以了：

```cpp
// winMain.hpp
#pragma once
#include <wx/wx.h>

class App : public wxApp
{
public:
	virtual bool OnInit() override;
	virtual int OnExit() override;
};
```

```cpp
// winMain.cpp
#include "winMain.hpp"

#include <wx/wx.h>

#include "lgcCore.hpp"

wxIMPLEMENT_APP(win::App);

bool App::OnInit()
{
	wxInitAllImageHandlers();

    // 初始化函数
	core::initializeCore(
		"Test",                 // 这里填 App 的名字，将会显示在主窗口上
		"./user_conf.json",     // 这里填用户修改过的配置的路径
		"./core_conf.json",     // 这里填内核配置的路径
		"./CorePlugins",        // 这里填存放核心插件的文件夹路径
		"./UserPlugins"         // 这里填存放用户插件的文件夹路径
	);

	core::UiManager::get().show();
	SetTopWindow(core::UiManager::get().getRootWindow());
	return true;
}

int App::OnExit()
{
    // 销毁函数
	core::shutdownCore();

	return 0;
}
```

## 1.2 组成

Core 由11个管理器组成，分别是

* 命令管理器（Command Manager）
* 配置管理器（Config Manager）
* 数据管理器（Data Manager）
* 事件管理器（Event Manager）
* 语言管理器（I18n Manager）
* 日志管理器（Log Manager）
* 插件管理器（Plugin Manager）
* 资源管理器（Resource Manager）
* 服务管理器（Service Manager）
* 任务管理器（Task Manager）
* 界面管理器（Ui Manager）

以及一些辅助函数（Utility），接下来我们一一简要地介绍他们的功能。

### 1.2.1 命令管理器

[命令管理器](2.%20命令管理器.md)管理命令和事务的运行、撤销和重做。

### 1.2.2 配置管理器（全局单例）

[配置管理器](3.%20配置管理器.md)管理内核、插件及用户自定义的配置。

### 1.2.3 数据管理器（全局单例）

[数据管理器](4.%20数据管理器.md)管理文档的创建、关闭、切换与获取。

### 1.2.4 事件管理器（全局单例）

[事件管理器](5.%20事件管理器.md)管理话题的订阅、发布和退订。

### 1.2.5 语言管理器（全局单例）

[语言管理器](6.%20语言管理器.md)管理语言的加载、切换。

### 1.2.6 日志管理器（全局单例）

[日志管理器](7.%20日志管理器.md)管理日志的启动、记录和关闭。

### 1.2.7 插件管理器（全局单例）

[插件管理器](8.%20插件管理器.md)管理插件的添加、删除、安装、卸载等功能。

### 1.2.8 资源管理器（全局单例）

[资源管理器](9.%20资源管理器.md)管理资源的加载和卸载。

### 1.2.9 服务管理器（全局单例）

[服务管理器](10.%20服务管理器.md)管理命名服务和非命名服务的注册、获取和注销。

### 1.2.10 任务管理器（全局单例）

[任务管理器](11.%20任务管理器.md)管理任务在后台的运行。

### 1.2.11 界面管理器（全局单例）

[界面管理器](12.%20界面管理器.md)管理界面的显示、ID的分配、控件的注册和布局的保存。
